Lists chapter notes:* List is a sequence of values* Values within a list is called elements and items* List within another list is considered nested* List values can be assigned to variables* “Unlike strings, lists are mutable”* Use a for loop to transverse the elements in a list* Lists have operations such as + and ** Lists are capable of being sliced* Different methods that can be operated on listso Append – adds a new element to the end of a listo Extend – takes a list as an argument and appends all of the elementso Sort – arranges the elements of the list from low to high*
