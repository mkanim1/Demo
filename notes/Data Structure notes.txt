Data Structure notes:* Data looks different on computers* Sentences are seen as numberso These numbers are used in Hex editors* A hex editor is a way of looking at data and memory* Hex editor representation is in hexadecimal *  Computers memory is a giant bank of switches* Data is based on Binary codes and the digits of 1&0 Only* Hexadecimal simplifies binary reading by adding letters* A Bit is a single Binary (1 or 0)* Byte = 8 bits* Word = Whatever the processor wants usually a power of 2* Most systems run 64 bit, soon enough we’ll run 128* Numbers on the left in hex editor represent memory location* Common data typeso Character – Char/chro Integer – into Floating point – float
